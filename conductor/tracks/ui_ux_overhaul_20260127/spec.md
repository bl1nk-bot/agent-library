# Specification: AI Command Hub Overhaul

## Core Aesthetic
- **Theme:** Digital Noir & Glass Foundry (Obsidian Black #050505).
- **Glassmorphism:** Backdrop blur for all panels.
- **Typography:** Inter (UI), JetBrains Mono (Code).

## Layout
- **Structure:** 3-Column Layout (Sidebar, Main Chat, Context Rack).
- **Scaling:** Fluid height (100vh) with independent column scrolling.

## Interactions
- **Framer Motion:** Micro-interactions (Scale on click, hover glow).
- **Transitions:** Smooth page fades and layout animations.
