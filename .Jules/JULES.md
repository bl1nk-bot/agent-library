# JULES (Joint Unified Logging and Execution Standard)

## Core Philosophy
1. Centralized logging for multi-agent collaboration.
2. Conflict prevention through active task tracking.
3. Standardized execution protocols for all autonomous agents.

## Protocols
- All agents must read and comply with this document.
- Tasks must be logged in .Jules/task-log.md.
- Guardian agents must follow the Guardian Skill guidelines.
